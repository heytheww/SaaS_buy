# 管理接口

1.管理用户

【url】 /manage/user
【POST】 新增用户
```
{
    "username":"user",
    "password":"123",
    "phone":"13200000000",
    "role":0,//0管理员 1普通用户
    "grade":1,//用户等级
}
{
    "data":{
        "id":"1",
    },
    "result":{
        "code":"200", 
        "message":"success"
    }
}
```

【DELETE】删除用户
```
{
    "id":"1",
}
{
    "data":{},
    "result":{
        "code":"200", 
        "message":"success"
    }
}
```

【PATCH】 修改用户
```
{
    "id":"1",
    "password":"123",
    "phone":"13200000000",
    "role":0,//0管理员 1普通用户
    "grade":1,//用户等级
}
{
    "data":{},
    "result":{
        "code":"200", 
        "message":"success"
    }
}
```

【GET】 查询用户
```
{
    "id":"1",
}
{
    "data":{
        "id":"1",
        "username":"user",
        "password":"123",
        "phone":"13200000000",
        "role":0,//0管理员 1普通用户
        "grade":1,//用户等级
        "create_time":"2020-01-35 18:23:55",
    },
    "result":{
        "code":"200", 
        "message":"success"
    }
}
```
